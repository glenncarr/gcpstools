# Contributing to gcpstools

## Repository structure

```
gcpstools/
├── .github/workflows/publish.yml   # CI/CD for PowerShell Gallery
├── src/gcpstools/
│   ├── Private/                    # Internal helper functions
│   ├── Public/                     # Exported functions
│   ├── gcpstools.psd1              # Module Manifest
│   └── gcpstools.psm1              # Root Module Script
├── tests/                          # Pester tests
├── build.ps1                       # Local build/test script
├── CHANGELOG.md
├── LICENSE
└── README.md
```

## Development

```powershell
# Import the module from source
Import-Module ./src/gcpstools -Force

# Run tests
./build.ps1 -Test
```

## Publishing

The published Gallery README is generated by copying the root `README.md` into
the module folder at publish time (see `build.ps1 -Publish`). Do not commit
`src/gcpstools/README.md` — it is git-ignored and produced on demand.
